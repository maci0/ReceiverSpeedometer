# ReceiverSpeedometer

Displays the raft speed on the Receiver screen in Raft. Includes a simple
speed text readout and a block gauge.

## Features
- Speed readout in meters per second.
- Block gauge that fills based on speed.
- Lightweight scanning and update loop.

## Installation
1. Install Raft Mod Loader.
2. Download the latest `ReceiverSpeedometer.rmod`.
3. Drop the file into your Raft mods folder.

## Usage
- Place a Receiver in your raft.
- The speed readout appears on the Receiver screen.

## Build (optional)
Linux/macOS:
```bash
./build.sh
```

Windows:
```bat
build.bat
```

Output: `ReceiverSpeedometer.rmod`
